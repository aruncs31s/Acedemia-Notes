# **Optimization Techniques**

## Syllabus

- [ ] **Module 1: Introduction to Classical Methods**
  - [ ] Engineering applications of optimization
  - [ ] Formulating design problems as mathematical programming
  - [ ] Classifying problems (linear, convex, non-convex)
  - [ ] Classical optimization for unconstrained (single & multivariable)
  - [ ] Constrained problems & KKT conditions

- [ ] **Module 2: Linear Programming Problems**
  - [ ] Mathematical formulation of LP problems
  - [ ] Solving using Graphical method
  - [ ] Solving using Simplex method

- [ ] **Module 3: Game Theory & Network Path Models**
  - [ ] 2-person zero-sum games, saddle points, Mini-Max & Maxi-Min Theorems
  - [ ] Graphical solutions (2xn and mx2 games), dominance property
  - [ ] Network trees & Minimal Spanning Tree (Prim's Algorithm)
  - [ ] Shortest path problems (Dijkstra's Method)

- [ ] **Module 4: Nonlinear Unconstrained Optimization**
  - [ ] Single-variable: Fibonacci search method
  - [ ] Single-variable: Newton Raphson method
  - [ ] Multi-variable: Hook-Jeeves pattern search method
  - [ ] Multi-variable: Cauchy's (steepest descent) method

- [ ] **Module 5: Modern Methods of Optimization**
  - [ ] Genetic Algorithms (GA) - encoding, crossover, selection, mutation
  - [ ] Fuzzy logic - fuzzy sets, membership functions, operations on fuzzy sets
  - [ ] Optimization of fuzzy systems
