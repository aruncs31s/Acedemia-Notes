## Formula Quick Reference

| Topic | Formula |
|-------|---------|
| Friis | $P_r = P_t G_t G_r (\lambda/4\pi d)^2$ |
| Path Loss (dB) | $20\log_{10}(4\pi d/\lambda)$ |
| Doppler | $f_d = v/\lambda$ |
| Coherence Time | $T_c \approx 0.423/f_m$ |
| Coherence BW | $B_c \approx 1/(5\sigma_\tau)$ |
| Rayleigh Outage | $P_{out} = 1 - e^{-\gamma_{th}/\bar{\gamma}}$ |
| Shannon Capacity | $C = B\log_2(1+SNR)$ |
| BPSK BER (AWGN) | $P_b = Q(\sqrt{2E_b/N_0})$ |
| BPSK BER (Rayleigh) | $P_b = \frac{1}{2}(1 - \sqrt{\bar{\gamma}/(1+\bar{\gamma})})$ |
| PAPR | $PAPR = \max|x(t)|^2/E[|x(t)|^2]$ |
| Critical Freq | $f_{cr} = 9\sqrt{N_{max}}$ |
| MUF | $MUF = f_{cr} \sec(\phi_i)$ |
