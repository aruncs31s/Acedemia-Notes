# 4.3 Multiuser Systems

Multiuser systems allow multiple users to share the limited radio spectrum efficiently. This is essential for supporting many users in cellular networks, Wi-Fi, and other wireless systems.

## Uplink vs Downlink

### Downlink (Base Station to User)

- **Direction**: Base Station → Multiple Users
- **Challenges**:
  - Base station transmits to all users simultaneously
  - Requires coordination to prevent interference
  - Resources allocated by base station
- **Techniques**: FDMA, TDMA, CDMA, OFDMA all applicable

### Uplink (User to Base Station)

- **Direction**: Multiple Users → Base Station
- **Challenges**:
  - Multiple users transmit independently
  - Requires synchronization
  - Near-far problem: closer users have stronger signal
- **Techniques**: Same multiple access methods but more challenging

## Frequency-Division Multiple Access (FDMA)

FDMA assigns each user a unique frequency band.

### How It Works

- Total bandwidth W is divided into N frequency channels
- Each user gets exclusive use of their frequency band
- Guard bands between channels prevent overlap

### Advantages

- Simple implementation
- No complex synchronization required
- Continuous transmission possible
- Low interference between users (with proper guard bands)

### Disadvantages

- Inefficient: Resources wasted if user is idle
- Limited capacity: Number of users = Bandwidth / Channel width
- Requires good filter design to prevent adjacent channel interference

### Applications

- Old analog cellular systems (1G): AMPS, NMT
- Satellite communication
- Radio broadcasting (FM/AM)

### Capacity

- Channel capacity: C = B × log₂(1 + S/N)
- Number of users: N ≤ Total Bandwidth / Channel Bandwidth

## Time-Division Multiple Access (TDMA)

TDMA assigns each user a unique time slot.

### How It Works

- Time is divided into frames
- Each frame contains N time slots
- Each user transmits only during their assigned slot
- Users share the same frequency band

### Frame Structure

```
Frame:
| Slot 1 | Slot 2 | ... | Slot N | Guard |
|--------|--------|-----|--------|-------|
User 1   User 2         User N
```

### Advantages

- Higher capacity than FDMA
- Dynamic allocation possible
- Can combine with FDMA (FDMA/TDMA)
- Easy to implement digital transmission

### Disadvantages

- Requires strict synchronization
- Guard time needed to prevent overlap
- Burst transmission (not continuous)
- Delay proportional to number of users

### Applications

- GSM (2G): Combined with FDMA
- Digital cellular systems
- Satellite communication (TDMA)

### Efficiency

- Frame efficiency: (Frame time - Guard time) / Frame time
- Voice activity: TDMA can leverage voice activity detection

## Code-Division Multiple Access (CDMA)

CDMA assigns each user a unique spreading code.

### How It Works

- All users transmit simultaneously on the same frequency
- Each user has a unique orthogonal (or nearly orthogonal) code
- Receiver uses the code to separate users
- Processing gain determines number of users

### Spreading

- **Spreading**: Multiply data by high-rate spreading code
- **Chip rate**: Much higher than data rate
- **Processing Gain (PG)**: Ratio of chip rate to data rate

```
User 1: Data1 × Code1 → Spread Signal 1
User 2: Data2 × Code2 → Spread Signal 2
...
All signals added → Transmitted over air
```

### Types of Codes

#### Orthogonal Codes (Walsh Codes)

- Perfectly orthogonal: Inner product = 0
- Used for downlink (all signals from same source)
- Example: Hadamard matrices
- Problem: Need strict synchronization

#### Pseudo-Noise (PN) Sequences

- Nearly orthogonal properties
- Used for uplink
- Long period PN sequences
- Requires power control

### Advantages

- Soft capacity: More users with graceful degradation
- Frequency reuse factor = 1 (all cells can use same frequency)
-抗干扰能力强 (抗窄带干扰)
- No need for strict time synchronization
- Built-in diversity (through spreading)

### Disadvantages

- Near-far problem: Strong signal from one user blocks others
- Requires power control
- Complex receivers
- Lower spectral efficiency than OFDMA

### Applications

- 3G cellular: cdma2000, WCDMA
- GPS
- UMTS

### Capacity

- Maximum users = Processing Gain / (Signal to Interference Ratio)
- Capacity increases with power control
- Near-far problem limits capacity without power control

## Orthogonal Frequency-Division Multiple Access (OFDMA)

OFDMA combines OFDM with multiple access by assigning subcarriers to users.

### How It Works

- Total bandwidth divided into many subcarriers
- Subcarriers grouped into subchannels
- Each user gets one or more subchannels
- Subcarriers within subchannel are orthogonal

### Subcarrier Assignment

```
Subcarriers: | 1 | 2 | 3 | 4 | 5 | 6 | 7 | 8 | ... | N |
User 1:     | ✓ | ✓ |   |   |   | ✓ | ✓ |   |     |
User 2:     |   |   | ✓ | ✓ |   |   |   | ✓ |     |
```

### Types of Subcarrier Allocation

#### Distributed Subcarriers

- Subcarriers spread across entire bandwidth
- Provides frequency diversity
- Better for mobile users

#### Localized Subcarriers

- Consecutive subcarriers assigned
- Enables frequency-selective scheduling
- Better for fixed/nomadic users

### Advantages

- High spectral efficiency
- Handles frequency-selective fading well
- Simple equalization (FFT-based)
- Flexible resource allocation
- Low ISI (orthogonal subcarriers)
- Easy to implement with FFT

### Disadvantages

- High PAPR (Peak-to-Average Power Ratio)
- Sensitive to frequency offset
- Requires strict synchronization
- Complex base station

### Applications

- 4G LTE
- Wi-Fi (802.11)
- WiMAX
- 5G NR

### Resource Allocation

- Time-frequency grid
- Adaptive modulation and coding
- Frequency-selective scheduling
- Dynamic resource allocation

## Comparison of Multiple Access Techniques

| Feature | FDMA | TDMA | CDMA | OFDMA |
|---------|------|------|------|-------|
| Complexity | Low | Medium | High | High |
| Flexibility | Low | Medium | High | Very High |
| Capacity | Low | Medium | High | Very High |
| Sync Required | No | Yes | Yes (power) | Yes |
| Frequency Diversity | No | Limited | Yes | Yes |
| PAPR | Low | Low | Low | High |
| ICI | Low | Low | High | Very Low |
| Best For | Analog | Voice | 3G | 4G/5G |

## Multiple Access in Cellular Systems

### Generation Evolution

- **1G**: FDMA (AMPS, NMT)
- **2G**: TDMA/FDMA (GSM)
- **3G**: CDMA (WCDMA, cdma2000)
- **4G**: OFDMA
- **5G**: OFDMA + new multiple access (NOMA)

### Future: NOMA (Non-Orthogonal Multiple Access)

- Superposition of multiple users in power domain
- Successive interference cancellation at receiver
- Higher capacity than OFDMA
- Under research for 5G and beyond
